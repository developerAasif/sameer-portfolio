# Portfolio Setup Guide

## Installation

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Open** [http://localhost:3000](http://localhost:3000)

## Customization

### Update Your Information

1. **Personal Details:**
   - Edit `src/components/hero.tsx` - Update name and title
   - Edit `src/components/about.tsx` - Update about section
   - Edit `src/components/experience.tsx` - Update work experience
   - Edit `src/components/projects.tsx` - Update your projects
   - Edit `src/components/contact.tsx` - Update contact email
   - Edit `src/components/footer.tsx` - Update social links

2. **Skills:**
   - Edit `src/components/skills.tsx` - Update skills and proficiency levels

3. **Theme:**
   - Colors are configured in `src/app/globals.css`
   - Theme toggle is in the navbar (top right)

4. **Metadata:**
   - Edit `src/app/layout.tsx` - Update SEO metadata

## Features

✅ **Modern UI** - Built with shadcn/ui and Tailwind CSS  
✅ **Dark/Light Mode** - Theme toggle with smooth animations  
✅ **Smooth Animations** - Powered by Motion.dev

✅ **SEO Optimized** - Ready for production  
✅ **Type Safe** - Full TypeScript support

- **Next.js 16** - React framework

- **shadcn/ui** - UI components
